**Linux Commands to be used :**



**Linux File System**



1\) home/ : Contains the home directories of users. Example: /home/john



2\) etc/ : Stores system configuration files.



3\) var/ : Stores variable data such as logs, cache, mail, and spool files.



4\) bin/ : Contains essential user command binaries like ls, cp, mv, and cat.



**Basic Linux Commands**



1\) pwd : print working directory



2\) ls : Lists all files and directories in the current directory.



3\) ls-l : Displays files and directories in long format (permissions, owner, size, date, etc.)



4\) ls-a : Shows all files, including hidden files (files beginning with .)



**File operations in Linux :**



1. touch : create a new file



2\. mkdir : create a new directory



3\. cp : used to copy a file or directory.



4\. mv : Moves or renames a file or directory.



5\. nano.filename :  Opens or creates a file in the Nano text editor for editing.

for ex: nano new.txt means you will enter into text file and do write operations.

&#x20;

6\. cat.filename : Displays the contents of a file for eg: "hello Linux"



7\. ctrl M-B : inside the text file to take a backup of the written text file



8\. ctrl + X : to close the .txt text file 



9\. ctrl + C : to cancel the .txt text file



10\. rm : to delete and remove the file from any folder 



11\. rmdir : to delete a directory



12\. head : to show the first few lines of the written text in .txt file or any other file created in Linux



13\. head n-3 filename : to specifically print the starting 3 or 4 lines of the paragraph in any file



14\. tail : to print the last commands in the text 



15\. ctrl + O : Saves the file while using Nano.



16\. ctrl + G : Opens the Nano help menu.



17\. ctrl + K : Cuts the current line in Nano.



18\. ctrl + U : Pastes the cut line in Nano.



19\. ctrl + W : Searches for text in Nano.



20\. ctrl + \\ : Searches and replaces text in Nano.



21\. tail n-5 : Displays the last 5 lines of a file.
